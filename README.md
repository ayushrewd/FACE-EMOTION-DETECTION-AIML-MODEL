# Face Emotion Detection System

Real-time face emotion detection using Python and Computer Vision.

## Features
- Live webcam face detection
- Emotion classification (happy, sad, surprise, neutral)
- Confidence percentage
- HUD overlay using OpenCV

## Tech Stack
- Python
- OpenCV
- FER
- TensorFlow / Keras

## How to Run
pip install opencv-python fer tensorflow keras mtcnn  
python emotion.py
